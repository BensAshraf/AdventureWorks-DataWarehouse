# SSIS ETL Process

This ETL process was designed using SQL Server Integration Services (SSIS) concepts.

Packages:
- Bronze_Load
- Silver_Transform
- Gold_Load

Process:
1. Extract data from AdventureWorks2022
2. Clean and transform data
3. Load dimensional model into Gold schema
